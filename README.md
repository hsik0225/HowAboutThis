## Usage
1. OAuth2를 이용한 회원가입
2. 설정한 기간동안 먹지 않은 메뉴들 중에서 이용자들이 가장 선호하는 메뉴 순서대로 추천해줍니다
3. 메뉴 별로 선호도 통계를 내고 그래프로 보여줍니다
